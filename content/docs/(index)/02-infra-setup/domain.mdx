## 도메인(Domain)이란?

> 도메인은 집 주소입니다. IP 주소(예: 192.168.1.1)가 GPS 좌표라면, 도메인(예: myapp.com)은 "서울시 강남구 ○○빌딩"처럼 사람이 기억하기 쉬운 주소입니다.

## 환경변수(Environment Variables)란?

> 환경변수는 자물쇠의 비밀번호입니다. 코드에 직접 적으면 안 되는 API 키 같은 민감한 정보를 안전하게 보관하는 방법입니다.
