## 인프라란?

> 건물을 짓기 전에 땅을 고르고 기초 공사를 하듯, 제품을 만들기 전에도 코드를 저장할 곳, 세상에 공개할 방법을 먼저 준비해야 합니다. 이것이 인프라(Infrastructure)입니다.

## 이 섹션에서 배우는 것

1. **GitHub** — 코드를 안전하게 저장하고 공유하는 곳
2. **Vercel** — 만든 것을 세상에 자동으로 공개하는 서비스
3. **도메인 & 환경변수** — 주소와 비밀 설정값
